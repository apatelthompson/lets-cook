# Let's Cook

**A hands-on workshop for complete beginners — from zero experience to your first app, live on the internet.**

---

## Why This Exists

Not everyone needs to become a chef. But everyone needs to be able to feed themselves.

AI is increasingly not just a tool — it's labor. Programmable labor. And the language you use to direct that labor is code. Not necessarily complex code. But enough to be in the room. Enough to speak the language.

In the same way that people learn English not because they love English, but because it became the lingua franca — code is becoming the language of work. If you don't speak it at all, you're not in the conversations about how work gets done.

This workshop is about learning to cook something real. Not becoming a chef. Just: enough to feed yourself.

**By the end:** you'll have built and deployed a web app that anyone in the world can visit. No prior experience needed.

---

## What You'll Build Toward

You'll choose one of four dishes to cook:

- **A personal portfolio page** — who you are, what you're about, your links
- **A quiz or trivia game** — questions, answers, score tracking
- **A countdown timer** — counting down to something you actually care about
- **A landing page for your idea** — if you've had something brewing, this is the day you build the page for it

## Before the Workshop — Install These First

**This is homework. Do it before you arrive.**

If you show up without these installed, you'll spend the first 20 minutes on setup while everyone else is cooking.

### 1. Cursor
Your kitchen. A code editor with AI built in. Free.

**→ Download at [cursor.com](https://cursor.com)**

After installing: open it, create a free account, and make sure it opens without errors.

### 2. Claude Code
Your sous chef. The AI you'll work with throughout the workshop.

**→ Install by opening a terminal and running:**
```
npm install -g @anthropic/claude-code
```

Then run `claude` and log in with an Anthropic account. Create one free at [claude.ai](https://claude.ai) if you don't have one.

**Not sure if Node/npm is installed?** Run `node --version` in your terminal. If you get a number back, you're good. If not, install Node first at [nodejs.org](https://nodejs.org) (get the LTS version).

### 3. This project folder
Download or clone this repo to your computer. Put it somewhere you'll find it — your Desktop works fine.

## On the Day

1. Open **Cursor**
2. Go to **File → Open Folder** → select this `lets-cook` folder
3. Press **`Cmd+L`** (Mac) or **`Ctrl+L`** (Windows) to open the chat panel
4. Type anything — "ready to start" or just "hi"

Claude will read the instructor guide automatically and begin walking you through the workshop.
