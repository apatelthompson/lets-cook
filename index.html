The philosophy

Learning to code is like learning to cook.

🍳

Not everyone needs to be a chef.

But everyone needs to be able to feed themselves. You don't need to become a software engineer — you just need to be able to cook something real.

🌐

Code is the new lingua franca.

In the same way people learned English because it became the language of business — code is becoming the language you need to participate in how work works. If you're under 40, this is not optional.

🤖

AI is programmable labor.

The language you use to program it is code. If you don't speak it, you won't be part of the conversations about how work gets done — or whether you want a machine doing something on your behalf.

"You need to understand the kitchen — the tools, the appliances, the layout, what's safe and what's hard to come back from."
